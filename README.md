# Rock Paper Scissors – Python Game

This is a simple and fun implementation of the classic **Rock-Paper-Scissors** game in Python. The player competes against the computer, which makes random choices.This project is a Python-based implementation of the classic Rock Paper Scissors game, designed with clean logic and structured code practices. The game allows a user to compete against the computer, which makes its selection using Python's built-in randomization module. The program evaluates the choices and determines the result based on the traditional rules of the game.

This project demonstrates foundational Python concepts, including:

Conditional statements

Input handling

Random module usage

Basic program flow and decision-making

It is a beginner-friendly project suitable for those starting their journey in Python programming, and it serves as a simple example of interactive console-based applications.

---

##  Game Rules
- Rock vs Scissors → **Rock wins**
- Scissors vs Paper → **Scissors wins**
- Paper vs Rock → **Paper wins**
- Same choice → **Draw**
